Alerts

Alerts are used to attract user's attention for important information without interrupting the user's flow.
There are 4 types of alert variants available: Primary | Success | Error | Warning.
Alert can be created by using class alert and then using the class name for the specific class.

Primary! This is a Primary Alert !
Success! This is a Success Alert !
Error! This is a Error Alert !!!
Warning! This is a warning Alert !!!!!
